# Gold Market Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes gold market trends and performance using interactive visualizations. The dashboard helps users understand price movements, market volatility, trading volume, and monthly returns.

## Features

- Gold Price Trend Analysis
- High vs Low Price Comparison
- Daily Return Waterfall Analysis
- Trading Volume Analysis
- Volatility Analysis
- Interactive Year and Month Filters

## Tools Used

- Power BI
- DAX
- Data Modeling
- Data Visualization
